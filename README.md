# NACHA Parser

A Python library for parsing and processing NACHA (National Automated Clearing House Association) files.

## Overview

This project provides tools to read, parse, and validate NACHA files used in ACH (Automated Clearing House) banking transactions.

## Installation

```bash
pip install -e .
```

## Development Setup

Install development dependencies:

```bash
pip install -e ".[dev]"
```
